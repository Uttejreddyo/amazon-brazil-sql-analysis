# Amazon Brazil E-Commerce SQL Data Analysis

## Business Objective
Analyze Amazon Brazil e-commerce data to derive operational, revenue, customer, and delivery insights using SQL, and support data-driven business decisions.

## Dataset Overview
- Orders, order items, payments, customers, sellers, and delivery data
- Multi-table relational dataset representing real-world e-commerce operations

## Tools Used
- SQL (PostgreSQL / MySQL-style queries)
- Python (minor usage for data handling, where applicable)

## Key Analysis Areas
- Revenue contribution by product categories
- Payment method preferences and trends
- Order volume and seasonal patterns
- Delivery time analysis and delay identification
- Customer repeat behavior and lifetime value segmentation

## Key Insights
- Identified high-revenue product categories and peak demand periods
- Highlighted dominant payment methods and regional preferences
- Exposed delivery delays impacting customer retention
- Segmented customers based on repeat purchases and delivery experience

## Project Artifacts
- **SQL Script:** Complete SQL queries from table creation to analysis (`SQL/`)
- **Analysis PDF:** Question-wise SQL execution with outputs and explanations (`Analysis_Documentation/`)

## Outcome
This project demonstrates strong SQL skills, relational data understanding, and the ability to translate complex query results into actionable business insights.
